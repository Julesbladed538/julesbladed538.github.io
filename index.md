3. An active Baidu Netdisk account.

## ⚙️ Installation Process

5. Launch the application from your desktop shortcut.

## 🔗 Connecting to Baidu Netdisk

The application requires permission to access your cloud files.

1. Open the application settings menu.
2. Click the button labeled "Link Account."
3. The program opens your web browser.
4. Log in to your Baidu Netdisk account if you are not already signed in.
5. Click "Authorize" or "Allow" when the browser asks for permission.
6. Return to the application. It now recognizes your account.

## 📦 How to Transfer Files

Moving files involves a simple drag-and-drop system.

### Sending files to the cloud
1. Select the file or folder on your computer.
2. Drag the item into the main window of the application.
3. The transfer starts automatically.
4. A progress bar shows the status of your upload.

### Downloading files from the cloud
1. Look at the file list inside the application window.
2. Find the file you want to download.
3. Right-click the file name.
4. Choose the "Download" option.
5. Pick a folder on your computer where the file should go.

## 📈 Understanding the Tools

This application combines two powerful technologies to handle your data:

Bypy acts as the bridge between your computer and your cloud account. It handles the identification process so that Baidu Netdisk knows you are the owner of the files.

Aria2c manages the speed of your transfers. It breaks large files into smaller pieces. This makes the transfer flow faster and helps the process resume if your internet connection drops for a moment.

## 🔧 Troubleshooting

If the application fails to transfer a file, try these common fixes:

* Check your internet connection status.
* Restart the application.
* Refresh your account authorization in the settings menu.
* Check if your Baidu Netdisk cloud storage is full.

## 🔒 Security and Privacy

Your login credentials never leave your machine. The application stores only the authorization tokens required to move your files. It cannot access your passwords, emails, or personal information outside of your cloud file storage.
